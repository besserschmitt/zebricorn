# Zebricorn Consulting // Enterprise Streamlit Blueprint 🦄

This repository contains the source code for the **Zebricorn Consulting** digital presence. Built with Python and Streamlit, it serves as a lightweight, cloud-native showcase of technical expertise in the intersection of **Financial Business Logic** and **Data Architecture**.

## 🎯 Project Purpose
As a Senior Solution Consultant specializing in **Regulatory Reporting (Basel/AnaCredit)** and **SQL/ETL Architecture**, I use this project to demonstrate:
- **Cloud-Native Deployment:** Automated hosting via Streamlit Cloud.
- **Containerized Development:** Standardized environments using Docker/DevContainers.
- **Bilingual UI Logic:** Dynamic language switching (SE/EN) for international stakeholders.

## 🛠 Technical Stack
- **Framework:** [Streamlit](https://streamlit.io/) (Python-based UI)
- **Environment:** VS Code DevContainers (Docker)
- **Architecture:** Responsive, single-page application with modular translation logic.

## 🐳 Development Environment
This project is configured with a `.devcontainer/` setup. This ensures a consistent development environment by automatically configuring:
- Python 3.x
- Required VS Code extensions
- Pre-installed dependencies via `requirements.txt`

To run this project locally in VS Code:
1. Open the folder in VS Code.
2. Click **"Reopen in Container"** when prompted.
3. Run `streamlit run app.py` in the terminal.
